# Amino-Acid-Sequencer

A standalone futuristic diagnostic terminal built with HTML, CSS, and JavaScript.

## Features
- Validates DNA strands using the allowed bases A, C, T, and G
- Auto-upcases lowercase input
- Computes total molecular mass from the provided base weights
- Detects mutation-key matches inside the strand
- Shows a clear warning for invalid mutation keys
- Includes a polished sci-fi terminal-style UI

## Run locally
Open `index.html` in your browser.
